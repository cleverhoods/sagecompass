## PRAL (Perceive, Reason, Act, Learn) - cycle
- It's the AI Agent cognitive cycle:
  - `perceive()` – gather user input + context (MCP/Knowledge).
  - `reason()` – interpret, plan, and prepare structured intent.
  - `act()` – call tools or LLM to generate output.
  - `learn()` – validate, log, and refine for next run.
- Sources:
  - https://www.kellton.com/kellton-tech-blog/ai-enterprise-agentic-ai-delivers-real-time-business-analytics
  - https://assets.kpmg.com/content/dam/kpmgsites/in/pdf/2025/10/agentic-ai-the-future-of-autonomous-intelligence.pdf
  - https://medium.com/agenticai-the-autonomous-intelligence/agentic-ai-architecture-a-practical-production-ready-guide-2b2aa6d16118
